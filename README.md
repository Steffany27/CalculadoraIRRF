# Calculadora IRRF

Este projeto calcula o **INSS**, o **salário base** e o **IRRF** para um salário bruto informado, seguindo as regras da tabela de imposto de renda brasileira.

---

## Como rodar o projeto

### Opção 1: Rodar no navegador (sem instalar nada)
- Use [https://dotnetfiddle.net/](https://dotnetfiddle.net/)  
- Crie um novo projeto do tipo **Console**  
- Copie e cole os códigos dos arquivos:
  - `INSS.cs`
  - `IRRF.cs`
  - `FolhaPagamento.cs`
  - `Program.cs`  
- Clique em **Run** para executar o programa diretamente no navegador.

### Opção 2: Rodar localmente
